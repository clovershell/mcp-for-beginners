# ಈ ಉದಾಹರಣೆಯನ್ನು ಚಾಲನೆ ಮಾಡುವುದು

ಇದು LLM ಕ್ಲಯಿಂಟ್ ಸ್ನಾಡಿಗಾಗಿ ರಸ್ಟ್ ಪರಿಹಾರವಾಗಿದೆ. ನೀವು ರಸ್ಟ್ ಟೂಲ್ಚೈನ್ ಸ್ಥಾಪಿಸಿರುವುದು ಅಗತ್ಯ; [ಅಧಿಕೃತ ಸ್ಥಾಪನೆ ಮಾರ್ಗದರ್ಶಿ](https://www.rust-lang.org/tools/install) ನೋಡಿರಿ.

ಕ್ಲಯಿಂಟ್ ಗಿಥಬ್ ಮಾದರಿಗಳು ಇನ್ಫರೆನ್ಸ್ ಎಂಡ್ಪಾಯಿಂಟ್ (`https://models.github.ai/inference/chat`) ಮೂಲಕ ಮಾದರಿಯನ್ನು ಕರೆಸುತ್ತಿದ್ದು, ನಿಮ್ಮ GitHub ವೈಯಕ್ತಿಕ ಪ್ರವೇಶ ಟೋಕನ್ (PAT) ಅನ್ನು `OPENAI_API_KEY` ಪರಿಸರಚರದಿಂದ ಓತ್ತದೆ.

> [!NOTE]
> ಈ ರೆಪೋದಲ್ಲಿನ ಇತರೆ ಪರಿಹಾರಗಳು `GITHUB_TOKEN` ನ್ನು ಬಳಸುತ್ತವೆ. ರಸ್ಟ್ ಗೆ, OpenAI ಕ್ಲಯಿಂಟ್ ಸಂರಚನೆಗೆ సరಿಹೊಂದಿಸಲು `OPENAI_API_KEY` ಅನ್ನು ಇದೇ ಮೌಲ್ಯಕ್ಕೆ ಹೊಂದಿಸಬೇಕು.

## -0- ನಿಮ್ಮ GitHub ಟೋಕನ್ ಅನ್ನು ಸೆಟ್ ಮಾಡಿ

```bash
# zsh/bash
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# ಪವರ್‌ಶೆಲ್
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- ಉದಾಹರಣೆಯನ್ನು ರಚಿಸಿ

```bash
cargo build
```

## -2- ಉದಾಹರಣೆಯನ್ನು ಚಾಲನೆ ಮಾಡು

```bash
cargo run
```

ಕ್ಲಯಿಂಟ್ ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್ MCP ಸರ್ವರ್ ಅನ್ನು ಆರಂಭಿಸಿ, ಅದರ ಟೂಲ್ ಲಿಸ್ಟ್ ಅನ್ನು ತೆಗೆದುಕೊಂಡು, ಮಾದರಿ (`openai/gpt-5-mini`) ಬಳಸಿ `add` ಟೂಲ್ ಅನ್ನು ಕರೆಸುತ್ತದೆ. ಟೂಲ್ ಕರೆ (ಉದಾಹರಣೆಗೆ, "Calling tool: add") ಮತ್ತು ಆ ಕರೆ ಫಲಿತಾಂಶವನ್ನು ನೀವು ನೋಡಬಹುದು.
