# MCP ನಲ್ಲಿ ಉನ್ನತ ವಿಷಯಗಳು

[![ಉನ್ನತ MCP: ಸುರಕ್ಷಿತ, ವಿಸ್ತಾರಗೊಳ್ಳುವ ಮತ್ತು ಬಹುಮಾಧ್ಯಮ AI ಏಜೆಂಟ್ಗಳು](../../../translated_images/kn/06.42259eaf91fccfc6.webp)](https://youtu.be/4yjmGvJzYdY)

_(ಈ ಪಾಠದ ವೀಡಿಯೋವನ್ನು ನೋಡುವುದಕ್ಕಾಗಿ ಮೇಲಿರುವ ಚಿತ್ರವನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ)_

ಈ ಅಧ್ಯಾಯವು ಮಾದರಿ প্রসಂಗ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಅನುಷ್ಠಾನದ ಕೆಲವು ಉನ್ನತ ವಿಷಯಗಳನ್ನು ಆವರಿಸಿದೆ, ಇದರಲ್ಲಿ ಬಹುಮಾಧ್ಯಮ ಇಂಟಿಗ್ರೇಷನ್, ವಿಸ್ತಾರಗೊಳ್ಳುವಿಕ, ಸುರಕ್ಷತೆ ಉತ್ತಮ ಓಟಗಳು, ಮತ್ತು ಎಂಟರ್ಪ್ರೈಸ ಇಂಟಿಗ್ರೇಷನ್ ಸೇರಿವೆ. ಈ ವಿಷಯಗಳು ಆಧುನಿಕ AI ವ್ಯವಸ್ಥೆಗಳ ಬೇಡಿಕೆಯನ್ನು ಪೂರೈಸಬಹುದಾದ ಬಲಿಷ್ಠ ಮತ್ತು ಉತ್ಪಾದನಾ-ಬಯಸುವ MCP ಅಪ್ಲಿಕೇಶನ್ ರಚಿಸಲು ಬಹುಮುಖ್ಯ.

## ಅವಲೋಕನ

ಈ ಪಾಠವು ಮಾದರಿ প্রসಂಗ ಪ್ರೋಟೋಕಾಲ್ ಅನುಷ್ಠಾನದ ಉನ್ನತ ಪರಿಕಲ್ಪನೆಗಳನ್ನು ಪರಿಶೀಲಿಸುತ್ತದೆ, ಮುಖ್ಯವಾಗಿ ಬಹುಮಾಧ್ಯಮ ಇಂಟಿಗ್ರೇಷನ್, ವಿಸ್ತಾರಗೊಳ್ಳುವಿಕ, ಸುರಕ್ಷತೆ ಉತ್ತಮ ಓಟಗಳು, ಮತ್ತು ಎಂಟರ್ಪ್ರೈಸ ಇಂಟಿಗ್ರೇಷನ್ ಮೇಲೆ ಕೇಂದ್ರೀಕರಿಸುತ್ತದೆ. ಈ ವಿಷಯಗಳು ಸಂಕೀರ್ಣವಾದ ಅಗತ್ಯಗಳನ್ನು ನಿರ್ವಹಿಸಲು MCP ಅಪ್ಲಿಕೇಶನ್ಗಳನ್ನು ನಿರ್ಮಿಸಲು ಅಗತ್ಯ.

> **ಮುಂದಿನ ಕಡೆ ನೋಡಿದರೆ:** ಕೆಳಗಿನ ಕೆಲವು ವಿಷಯಗಳು `2026-07-28` MCP ಸ್ಪೆಸಿಫಿಕೇಷನ್ ಬಿಡುಗಡೆ ಅಭ್ಯರ್ಥಿಯಿಂದ ಪ್ರಭಾವಿತರಾಗಿವೆ — ರೂಟ್ প্রসಂಗಗಳು (5.4) ಮತ್ತು ಸಮ್ಪ್ಲಿಂಗ್ನು (5.6) ಬಿಡುಗಡೆ ಅಭ್ಯರ್ಥಿ ನಿಷೇಧಿಸಿದ ಪ್ರಿಮಿಟಿವ್ ಗಳ ಮೇಲ್ಪಡೆ ಮತ್ತು ಪ್ರೋಟೋಕಾಲ್ ವೈಶಿಷ್ಟ್ಯಗಳಲ್ಲಿ ಉಲ್ಲೇಖಿಸಿದ ಪ್ರಯೋಗಾತ್ಮಕ ಟಾಸ್ಕ್ಸ್ ವೈಶಿಷ್ಟ್ಯವು ವಿಶಿಷ್ಟ ಟಾಸ್ಕ್ಸ್ ವಿಸ್ತರಣೆಗೆ ಸ್ಥಳಾಂತರವಾಗಿದೆ. ವಿವರಗಳಿಗೆ [MCP ನಲ್ಲಿ ಏನು ಬದಲಾಗುತ್ತಿದೆ: 2026-07-28 ಬಿಡುಗಡೆ ಅಭ್ಯರ್ಥಿ](../01-CoreConcepts/mcp-2026-07-28-release-candidate.md) ನೋಡಿ.

## ಕಲಿಕೆಯ ಗುರಿಗಳು

ಈ ಪಾಠದ ಮುಗಿಸಿ, ನೀವು ಕೈಗೂಡಿಸಬಹುದಾಗಿದೆ:

- MCP ಚಟುವಟಿಕೆಗಳಲ್ಲಿ ಬಹುಮಾಧ್ಯಮ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಅನುಷ್ಠಾನಗೊಳಿಸುವುದು
- ಹೆಚ್ಚಿನ ಬೇಡಿಕೆಗೆ ತಕ್ಕ MCP ವಾಸ್ತುಶಿಲ್ಪಗಳನ್ನು ವಿನ್ಯಾಸಗೊಳಿಸುವುದು
- MCP ರ ಸುರಕ್ಷತಾ ತತ್ವಗಳಿಗೆ ಅನುಗುಣವಾಗಿ ಸುರಕ್ಷತಾ ಉತ್ತಮ ಓಟಗಳನ್ನು ಅನ್ವಯಿಸುವುದು
- ಎಂಟರ್ಪ್ರೈಸ AI ವ್ಯವಸ್ಥೆಗಳ ಮತ್ತು ಫ್ರೇಮ್‌ವರ್ಕ್‌ಗಳೊಂದಿಗೆ MCP ಅನ್ನು ಒಗ್ಗೂಡಿಸುವುದು
- ಉತ್ಪಾದನಾ ಪರಿಸರಗಳಲ್ಲಿ ಕಾರ್ಯಕ್ಷಮತೆ ಮತ್ತು ನಂಬಿಕೆಗುರುವಿಕೆಯನ್ನು ದೃಢೀಕರಿಸುವುದು

| [5.16 ಪ್ರೋಟೋಕಾಲ್ ವೈಶಿಷ್ಟ್ಯಗಳ ಗಾಢ ವಿಶ್ಲೇಷಣೆ](./mcp-protocol-features/README.md) | ಪ್ರೋಟೋಕಾಲ್ ವೈಶಿಷ್ಟ್ಯಗಳು | ಪ್ರಗತಿ ಸೂಚನೆಗಳು, ವಿನಂತಿ ರದ್ದುಪಡಿಸುವಿಕೆ, ಸಂಪನ್ಮೂಲ ಟೆಂಪ್ಲೇಟ್ಗಳು ಮತ್ತು ದೋಷ ನಿರ್ವಹಣಾ ಮಾದರಿಗಳನ್ನು ಒಳಗೊಂಡ ಉನ್ನತ ಪ್ರೋಟೋಕಾಲ್ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಮಾಸ್ಟರ್ ಮಾಡಿರಿ.|
| [5.17 ಪ್ರತಿಸ್ಪರ್ಧಾತ್ಮಕ ಬಹು-ಏಜೆಂಟ್ ಯುಕ್ತಿವಾದ](./mcp-adversarial-agents/README.md) | ಪ್ರತಿಸ್ಪರ್ಧಾತ್ಮಕ ಏಜೆಂಟ್ಗಳು | ವಿರೋಧಿ ಸ್ಥಾನಗಳನ್ನು ಹೊಂದಿರುವ ಇಬ್ಬರು ಏಜೆಂಟ್‌ಗಳನ್ನು ಬಳಸಿ, ಒಂದೇ MCP ಉಪಕರಣ ಸಂಗ್ರಹವನ್ನು ಹಂಚಿಕೊಂಡು, ಹಳ್ಳಸಿ ಗತಿಯು, ಎಡ್ಜ್ ಪ್ರಕರಣಗಳನ್ನು ಪತ್ತೆ ಹಚ್ಚಿ ಮತ್ತು ನಾವುಕಾದ ನೀತಿ ಸಂವಾದದಿಂದ ಉತ್ತಮ-ಅನುಗುಣಿತ ಫಲಿತಾಂಶಗಳನ್ನು ಉತ್ಪಾದಿಸಿರಿ.|

> **MCP ಸ್ಪೆಸಿಫಿಕೇಷನ್ 2025-11-25 ನಲ್ಲಿ ಹೊಸದಾಗಿ ಸೇರಿಸಿರುವುದು**: ಸ್ಪೆಸಿಫಿಕೇಷನ್ ಈಗ ಪ್ರಯೋಗಾತ್ಮಕ ಸಹಾಯವನ್ನು **ಟಾಸ್ಕ್ಗಳ** (ಪ್ರಗತಿ ಟ್ರ್ಯಾಕಿಂಗ್ ಜೊತೆಗೆ ದೀರ್ಘಕಾಲದ ಕಾರ್ಯಗಳು), **ಟೂಲ್ ಟಿಪ್ಪಣಿಗಳು** (ಸುರಕ್ಷತೆಗೆ ಉಪಕರಣ ವರ್ತನೆ ಬಗ್ಗೆ ಮೆಟಾಡೇಟಾ), **URL ಮೋಡ್ ಪ್ರೇರಣೆ** (ಗ್ರಾಹಕರಿಂದ ನಿರ್ದಿಷ್ಟ URL ವಿಷಯ ಕೇಳುವುದು) ಮತ್ತು ವೃದ್ಧಿಸಿದ **ರೂಟ್ಸ್** (ಕಾರ್ಯಸ್ಥಳ প্রসಂಗ ನಿರ್ವಹಣೆಗೆ) ಒಳಗೊಂಡಂತೆ ಒದಗಿಸುತ್ತದೆ. ಸಂಪೂರ್ಣ ವಿವರಗಳಿಗಾಗಿ [MCP ಸ್ಪೆಸಿಫಿಕೇಷನ್ ಬದಲಾವಣೆ ಮಾಹಿತಿ](https://spec.modelcontextprotocol.io/) ನೋಡಿ.

- [MCP ಡಾಕ್ಯುಮೆಂಟೇಶನ್](https://modelcontextprotocol.io/)
- [MCP ಸ್ಪೆಸಿಫಿಕೇಷನ್ (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/)
- [GitHub ರೆಪೊ](https://github.com/modelcontextprotocol)
- [OWASP MCP ಟಾಪ್ 10](https://microsoft.github.io/mcp-azure-security-guide/mcp/) - ಸುರಕ್ಷತಾ ಅಪಾಯಗಳು ಮತ್ತು ನಿಗ್ರಹಗಳು
- [MCP ಸುರಕ್ಷತಾ ಸಮಿಟ್ ಕಾರ್ಯಾಗಾರ (ಶೆರ್ಪಾ)](https://azure-samples.github.io/sherpa/) - ಪ್ರಾಯೋಗಿಕ ಸುರಕ್ಷತಾ ತರಬೇತಿ

## ಪ್ರಮುಖ ಒಗ್ಗಟ್ಟೆಗಳು

- ಬಹುಮಾಧ್ಯಮ MCP ಅನುಷ್ಠಾನಗಳು AI ಸಾಮರ್ಥ್ಯವನ್ನು ಪಠ್ಯ ಪ್ರಕ್ರಿಯೆ ಮೀರಿ ವಿಸ್ತರಿಸುತ್ತವೆ
- ವಿಸ್ತಾರಗೊಳ್ಳುವಿಕೆ ಎಂಟರ್ಪ್ರೈಸ ನಿಯೋಜನೆಗಳಿಗೆ ಅವಶ್ಯಕ ಮತ್ತು ಅದನ್ನು ಆವರ್ತಮಾನದ ಮೂಲಕ ಹಾಗೂ ಅಡ್ಡಾಗಿ ವಿಸ್ತಾರಗೊಳಿಸುವ ಮೂಲಕ ಸಾಧಿಸಬಹುದು
- ಸಮಗ್ರ ಸುರಕ್ಷತಾ ಕ್ರಮಗಳು ಡೇಟಾವನ್ನು ರಕ್ಷಿಸಿ ಮತ್ತು ಸರಿಯಾದ ಪ್ರವೇಶ ನಿಯಂತ್ರಣ ಒದಗಿಸುತ್ತವೆ
- ಅಜೂರ್ OpenAI ಮತ್ತು ಮೈಕ್ರೋಸಾಫ್ಟ್ AI ಫೌಂಡ್ರಿ వంటి ವೇದಿಕೆಗಳೊಂದಿಗೆ ಎಂಟರ್ಪ್ರೈಸ ಒಕ್ಕೂಟ MCP ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಹೆಚ್ಚಿಸುತ್ತದೆ
- ಉನ್ನತ MCP ಅನುಷ್ಠಾನಗಳು ಆಪ್ಟಿಮೈಸ್ ಮಾಡಿದ ವಾಸ್ತುಶಿಲ್ಪಗಳು ಮತ್ತು ಗಮನಪೂರ್ವಕ ಸಂಪನ್ಮೂಲ ನಿರ್ವಹಣೆಯಿಂದ ಲಾಭಪಡೆಯುತ್ತವೆ

## ವ್ಯಾಯಾಮ

ನಿರ್ದಿಷ್ಟ ಬಳಕೆಗೆ ಎಂಟರ್ಪ್ರೈಸ-ಗ್ರೇಡ್ MCP ಅನುಷ್ಠಾನವನ್ನು ವಿನ್ಯಾಸಗೊಳಿಸಿ:

1. ನಿಮ್ಮ ಬಳಕೆಯ ಬಹುಮಾಧ್ಯಮ ಅಗತ್ಯಗಳನ್ನು ಗುರುತಿಸಿ
2. ಸಂವೇದನಶೀಲ ಡೇಟಾವನ್ನು ರಕ್ಷಿಸಲು ಅಗತ್ಯವಿರುವ ಸುರಕ್ಷತಾ ನಿಯಂತ್ರಣಗಳನ್ನು ರೇಖಾಂಕನ ಮಾಡಿ
3. ಬದಲಾಗುವ ಭಾರದ ನಿಭಾಯಿಸಬಹುದಾದ ವಿಸ್ತಾರಗೊಳ್ಳುವ ವಾಸ್ತುಶಿಲ್ಪವನ್ನು ವಿನ್ಯಾಸ ಮಾಡಿ
4. ಎಂಟರ್ಪ್ರೈಸ AI ವ್ಯವಸ್ಥೆಗಳೊಂದಿಗೆ ಒಕ್ಕೂಟ ಬಿಂದುಗಳನ್ನು ಯೋಜಿಸಿ
5. ಸಾಧ್ಯವಿರುವ ಕಾರ್ಯಕ್ಷಮತೆ ಬಿಕ್ಕಟ್ಟುಗಳ ಮತ್ತು ಅವುಗಳನ್ನು ಪರಿಹರಿಸುವ ಯತ್ನಗಳನ್ನು ದಾಖಲೆ ಮಾಡಿ
