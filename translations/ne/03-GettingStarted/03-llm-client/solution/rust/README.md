# यो नमूना चलाउनुहोस्

यो LLM क्लाइन्ट नमुनाको लागि Rust समाधान हो। तपाईंले Rust टूलचेन स्थापना गर्न आवश्यक छ; हेर्नुहोस् [अधिकृत स्थापना गाइड](https://www.rust-lang.org/tools/install)।

क्लाइन्टले GitHub Models को इन्फरेन्स एन्डप्वाइन्ट (`https://models.github.ai/inference/chat`) मार्फत मोडललाई कल गर्दछ र तपाईंको GitHub व्यक्तिगत पहुँच टोकन (PAT) लाई `OPENAI_API_KEY` वातावरणीय भेरिएबलबाट पढ्दछ।

> [!NOTE]
> यस रिपोमा अन्य समाधानहरूले `GITHUB_TOKEN` प्रयोग गरेका छन्। Rust का लागि, OpenAI क्लाइन्ट कन्फिगरेशन मेल खान `OPENAI_API_KEY` लाई उही मानमा सेट गर्नुहोस्।

## -0- तपाईंको GitHub टोकन सेट गर्नुहोस्

```bash
# zsh/bash
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# पावरशेल
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- नमुना बनाउनुहोस्

```bash
cargo build
```

## -2- नमुना चलाउनुहोस्

```bash
cargo run
```

क्लाइन्टले क्याल्कुलेटर MCP सर्भर सुरु गर्छ, यसको उपकरण सूची ल्याउँछ, र मोडल (`openai/gpt-5-mini`) प्रयोग गरेर `add` उपकरण कल गर्छ। तपाईंले उपकरण कल देखाउने आउटपुट (जस्तै, "Calling tool: add") र उक्त कलको परिणाम देख्नुपर्नेछ।
