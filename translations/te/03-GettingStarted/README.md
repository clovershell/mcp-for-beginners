## ప్రారంభించడం  

[![మీ మొదటి MCP సర్వర్‌ను నిర్మించండి](../../../translated_images/te/04.0ea920069efd979a.webp)](https://youtu.be/sNDZO9N4m9Y)

_(ఈ పాఠం యొక్క వీడియోను చూడటానికి పై చిత్రం మీద క్లిక్ చేయండి)_

ఈ భాగంలో కొన్ని పాఠాలు ఉన్నాయి:

- **1 మీ మొదటి సర్వర్**, ఈ మొదటి పాఠంలో, మీరు మీ మొదటి సర్వర్‌ను ఎలా సృష్టించాలో మరియు ఇన్స్‌పెక్టార్ సాధనంతో దాన్ని పరిశీలించాలో, మీ సర్వర్‌ను పరీక్షించడానికి మరియు డీబగ్ చేయడానికి విలువైన ప్రక్రియను నేర్చుకుంటారు, [పాఠానికి](01-first-server/README.md)

- **2 కస్టమర్**, ఈ పాఠంలో, మీ సర్వర్‌కు కనెక్ట్ అయ్యే క్లయింట్‌ను ఎలా రాయాలో నేర్చుకుంటారు, [పాఠానికి](02-client/README.md)

- **3 LLMతో క్లయింట్**, క్లయింట్‌ని రాయడంలో ఇంకా మెరుగైన విధానం అది LLMను జోడించడం, తద్వారా అది మీ సర్వర్‌తో "నిగోషియేట్" చేసేందుకు వీలవుతుంది, [పాఠానికి](03-llm-client/README.md)

- **4 Visual Studio Codeలో సర్వర్ GitHub Copilot ఏజెంట్ మోడ్‌ను వినియోగించడం**. ఇక్కడ, మనం Visual Studio Code నుండి MCP సర్వర్‌ను నడుపుతున్నాం, [పాఠానికి](04-vscode/README.md)

- **5 stdio ట్రాన్స్‌పోర్ట్ సర్వర్** stdio ట్రాన్స్‌పోర్ట్ స్థానిక MCP సర్వర్-క్లయింట్ కమ్యూనికేషన్ కోసం పొందుపరిచిన ప్రామాణిక పద్ధతి, భద్రమైన subprocess-ఆధారిత కమ్యూనికేషన్‌ను మరియు అంతర్గత ప్రాసెస్ శాఖను అందిస్తుంది [పాఠానికి](05-stdio-server/README.md)

- **6 MCPతో HTTP స్ట్రీమింగ్ (Streamable HTTP)**. ఆధునిక HTTP స్ట్రీమింగ్ ట్రాన్స్‌పోర్ట్ గురించి తెలుసుకోండి (దూర MCP సర్వర్ల కోసం సూచించబడిన విధానం [MCP నిర్దిష్టత 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/basic/transports/#streamable-http)), పురోగతి నోటిఫికేషన్‌లు, మరియు Streamable HTTP ఉపయోగించి స్కేలబుల్, రియల్-టైం MCP సర్వర్లు మరియు క్లయింట్‌లు ఎలా అమలు చేయాలో. [పాఠానికి](06-http-streaming/README.md)

- **7 VSCode కోసం AI టూల్‌కిట్ వినియోగించడం** మీ MCP క్లయింట్‌లు మరియు సర్వర్‌లను పరీక్షించడానికి మరియు వినియోగించడానికి [పాఠానికి](07-aitk/README.md)

- **8 పరీక్షలు**. ఇక్కడ మనం ప్రధానంగా మన సర్వర్ మరియు క్లయింట్‌ను వివిధ విధాలలో ఎలా పరీక్షించాలో తెలుసుకుంటాం, [పాఠానికి](08-testing/README.md)

- **9 అమరిక**. ఈ అధ్యాయం మీ MCP పరిష్కారాలను అమలు చేసే వివిధ మార్గాలను చూడుతుంది, [పాఠానికి](09-deployment/README.md)

- **10 ప్రगतిశీల సర్వర్ వినియోగం**. ఈ అధ్యయం ప్రगतిశీల సర్వర్ వినియోగాన్ని కవర్ చేస్తుంది, [పాఠానికి](./10-advanced/README.md)

- **11 ధృవీకరణ**. ఈ అధ్యయం సులభమైన ధృవీకరణను ఎలా జోడించాలో చూస్తుంది, బేసిక్ ఆథ్ నుండి JWT మరియు RBAC ఉపయోగించడం వరకు. మీరు ఇక్కడ ప్రారంభించి, ఆపై అధునాతన విషయాలపై అధ్యాయం 5లో మరియు అనుబంధ భద్రతా సూచనల కోసం అధ్యాయం 2లో చూడాలని సిఫార్సు చేస్తాము, [పాఠానికి](./11-simple-auth/README.md)

- **12 MCP హోస్ట్లు**. Claude Desktop, Cursor, Cline, Windsurf వంటి ప్రాచుర్యం పొందిన MCP హోస్టు క్లయింట్‌లను కాన్ఫిగర్ చేసి ఉపయోగించండి. ట్రాన్స్‌పోర్ట్ రకాలు మరియు సమస్య పరిష్కారాలను తెలుసుకోండి, [పాఠానికి](./12-mcp-hosts/README.md)

- **13 MCP ఇన్స్పెక్టర్**. MCP ఇన్స్పెక్టర్ సాధన ఉపయోగించి మీ MCP సర్వర్‌లను పరస్పర చర్యలతో డీబగ్ చేయండి మరియు పరీక్షించండి. సాధనాలు, వనరులు మరియు ప్రోటోకాల్ సందేశాలను ఎలా పరిష్కరించాలో తెలుసుకోండి, [పాఠానికి](./13-mcp-inspector/README.md)

- **14 శాంప్లింగ్**. MCP సర్వర్‌లు MCP క్లయింట్‌లతో LLM సంబంధిత పనులపై కలిసి పనిచేసేందుకు సృష్టించండి (2026-07-28 విడుదల అభ్యర్థిలో అప్రచురితమైనది; 2025-11-25కి ఇంకా వర్తిస్తుంది). [పాఠానికి](./14-sampling/README.md)

- **15 MCP యాప్స్**. UI సూచనలతో కూడా స్పందించే MCP సర్వర్‌లను నిర్మించండి, [పాఠానికి](./15-mcp-apps/README.md)

మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) అనేది అనువర్తనాలు LLMలకు కాంటెక్స్ట్ అందించే విధానాన్ని ప్రామాణికీకరించే ఓపెన్ ప్రోటోకాల్. MCPని USB-C పోర్ట్‌లా ఆలోచించండి - ఇది AI మోడల్స్‌ను వివిధ డేటా మూలాలు మరియు సాధనాలకు కనెక్ట్ చేయడానికి ప్రామాణిక మార్గాన్ని అందిస్తుంది.

## అభ్యాస లక్ష్యాలు

ఈ పాఠం ముగిసే వరకు, మీరు చేయగలుగుతారు:

- C#, Java, Python, TypeScript, మరియు JavaScriptలో MCP అభివృద్ధి పరిసరాలను ఏర్పరచండి
- అనుకూల లక్షణాలతో (వనరులు, ప్రాంప్ట్లు, మరియు సాధనాలు) ప్రాథమిక MCP సర్వర్‌లను నిర్మించి, అమలు చేయండి
- MCP సర్వర్‌లకు కనెక్ట్ అయ్యే హోస్ట్ అనువర్తనాలను సృష్టించండి
- MCP అమలు చర్యలను పరీక్షించండి మరియు డీబగ్ చేయండి
- సాధారణ సెటప్ సవాళ్లు మరియు వాటి పరిష్కారాలను అర్థం చేసుకోండి
- మీ MCP అమలులను ప్రాచుర్యం పొందిన LLM సేవలకు కనెక్ట్ చేయండి

## మీ MCP పరిసరాన్ని సెటప్ చేయడం

MCPతో పనిచేయడం మొదలు పెట్టేముందు, మీ అభివృద్ధి పరిసరాన్ని సిద్ధం చేయడం మరియు ప్రాథమిక పని ప్రవాహాన్ని అర్థం చేసుకోవడం ముఖ్యము. ఈ భాగం మీకు సాఫీగా MCP ప్రారంభం కోసం ప్రారంభ సెటప్ దశలను గైడ్ చేస్తుంది.

### అవసరాలు

MCP అభివృద్ధిలోకి వెళ్లేముందు, మీరు కలిగి ఉండాలి:

- **అభివృద్ధి పరిసరం**: మీ ఎంచుకున్న భాష కోసం (C#, Java, Python, TypeScript, లేదా JavaScript)
- **IDE/ఎడిటర్**: Visual Studio, Visual Studio Code, IntelliJ, Eclipse, PyCharm, లేదా ఏ ఆధునిక కోడ్ ఎడిటర్
- **ప్యాకేజ్ మేనేజర్లు**: NuGet, Maven/Gradle, pip, లేదా npm/yarn
- **API కీలు**: మీరు మీ హోస్ట్ అనువర్తనాల్లో ఉపయోగించబోయే ఏ AI సేవల కోసం


### అధికారిక SDKలు

రాబోయే అధ్యాయాలలో మీరు Python, TypeScript, Java మరియు .NET ఉపయోగించి నిర్మించిన పరిష్కారాలు చూడగలుగుతారు. ఇక్కడ అన్ని అధికారికంగా మద్దతు పొందిన SDKలు ఉన్నాయి.

MCP అనేక భాషలకై అధికారిక SDKలను అందిస్తుంది (MCP నిర్దిష్టత 2025-11-25తో సమన్వయంగా):
- [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk) - Microsoftతో సహకారంలో నిర్వహించబడింది
- [Java SDK](https://github.com/modelcontextprotocol/java-sdk) - Spring AIతో సహకారంలో నిర్వహించబడింది
- [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) - అధికారిక TypeScript అమలు
- [Python SDK](https://github.com/modelcontextprotocol/python-sdk) - అధికారిక Python అమలు (FastMCP)
- [Kotlin SDK](https://github.com/modelcontextprotocol/kotlin-sdk) - అధికారిక Kotlin అమలు
- [Swift SDK](https://github.com/modelcontextprotocol/swift-sdk) - Loopwork AIతో సహకారంలో నిర్వహించబడింది
- [Rust SDK](https://github.com/modelcontextprotocol/rust-sdk) - అధికారిక Rust అమలు
- [Go SDK](https://github.com/modelcontextprotocol/go-sdk) - అధికారిక Go అమలు

## ముఖ్యమైన సారాంశాలు

- భాషకు ప్రత్యేకమైన SDKలతో MCP అభివృద్ధి పరిసరాన్ని సులభంగా సెటప్ చేయవచ్చు
- MCP సర్వర్‌లను నిర్మించడం అనగా స్పష్టమైన స్కీమాలతో సాధనాలను తయారుచేయడం మరియు నమోదు చేయడం
- MCP క్లయింట్‌లు సర్వర్‌లు మరియు మోడల్స్‌తో కనెక్ట్ అయి విస్తరించిన సామర్థ్యాలను వినియోగిస్తాయి
- MCP అమలుల కోసం పరీక్షించడం మరియు డీబగ్గింగ్ తప్పనిసరి
- అమరిక ఎంపికలు స్థానిక అభివృద్ధి నుండి క్లౌడ్ ఆధారిత పరిష్కారాల వరకు ఉంటాయి
