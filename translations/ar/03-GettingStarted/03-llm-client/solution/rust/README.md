# تشغيل هذا المثال

هذا هو حل Rust لعينة عميل LLM. تحتاج إلى تثبيت مجموعة أدوات Rust؛ انظر [الدليل الرسمي للتثبيت](https://www.rust-lang.org/tools/install).

يقوم العميل باستدعاء نموذج من خلال نقطة نهاية استدلال نماذج GitHub (`https://models.github.ai/inference/chat`) ويقرأ رمز الوصول الشخصي الخاص بك على GitHub (PAT) من متغير البيئة `OPENAI_API_KEY`.

> [!NOTE]
> تستخدم الحلول الأخرى في هذا الريبو `GITHUB_TOKEN`. بالنسبة لـ Rust، قم بتعيين `OPENAI_API_KEY` إلى نفس القيمة لتطابق إعدادات عميل OpenAI.

## -0- عيّن رمز GitHub الخاص بك

```bash
# زش/باش
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# باورشل
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- بناء العينة

```bash
cargo build
```

## -2- تشغيل العينة

```bash
cargo run
```

يبدأ العميل خادم MCP للحاسبة، ويجلب قائمة الأدوات الخاصة بها، ويستخدم النموذج (`openai/gpt-5-mini`) لاستدعاء أداة `add`. يجب أن ترى مخرجات تشير إلى استدعاء الأداة (مثلاً، "استدعاء الأداة: add") ونتيجة ذلك الاستدعاء.
