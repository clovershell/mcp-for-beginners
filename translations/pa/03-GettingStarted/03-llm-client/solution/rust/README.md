# ਇਸ ਸੈਂਪਲ ਨੂੰ ਚਲਾਉਣਾ

ਇਹ LLM ਕਲਾਇੰਟ ਸੈਂਪਲ ਲਈ ਰਸਟ ਹੱਲ ਹੈ। ਤੁਹਾਨੂੰ ਰਸਟ ਟੂਲਚੇਨ ਇੰਸਟਾਲ ਹੋਣਾ ਜਰੂਰੀ ਹੈ; ਵੇਖੋ [ਅਧਿਕਾਰਿਕ ਇੰਸਟਾਲ ਗਾਈਡ](https://www.rust-lang.org/tools/install).

ਕਲਾਇੰਟ ਮਾਡਲ ਨੂੰ GitHub ਮਾਡਲ ਇਨਫਰਨਸ ਏਂਡਪਾਇੰਟ (`https://models.github.ai/inference/chat`) ਰਾਹੀਂ ਕਾਲ ਕਰਦਾ ਹੈ ਅਤੇ ਤੁਹਾਡਾ GitHub ਨਿੱਜੀ ਪਹੁੰਚ ਟੋਕਨ (PAT) `OPENAI_API_KEY` ਵਾਤਾਵਰਣੀ ਵਾਲੀਅਬਲ ਤੋਂ ਪੜ੍ਹਦਾ ਹੈ।

> [!NOTE]
> ਇਸ ਰਿਪੋ ਵਿੱਚ ਹੋਰ ਹੱਲ `GITHUB_TOKEN` ਵਰਤਦੇ ਹਨ। ਰਸਟ ਲਈ, OpenAI ਕਲਾਇੰਟ ਸੰਰਚਨਾ ਨਾਲ ਮਿਲਾਓ ਕਰਨ ਲਈ `OPENAI_API_KEY` ਨੂੰ ਉਹੀ ਮੁੱਲ ਸੈੱਟ ਕਰੋ।

## -0- ਆਪਣਾ GitHub ਟੋਕਨ ਸੈੱਟ ਕਰੋ

```bash
# ਜ਼ੈਸ਼/ਬੈਸ਼
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# ਪਾਵਰਸ਼ੈੱਲ
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- ਸੈਂਪਲ ਬਣਾਓ

```bash
cargo build
```

## -2- ਸੈਂਪਲ ਚਲਾਓ

```bash
cargo run
```

ਕਲਾਇੰਟ ਕੈਲਕੂਲੇਟਰ MCP ਸਰਵਰ ਸ਼ੁਰੂ ਕਰਦਾ ਹੈ, ਆਪਣੀ ਔਜ਼ਾਰ ਸੂਚੀ ਲੈਦਾ ਹੈ, ਅਤੇ ਮਾਡਲ (`openai/gpt-5-mini`) ਦਾ ਉਪਯੋਗ ਕਰਕੇ `add` ਔਜ਼ਾਰ ਨੂੰ ਕਾਲ ਕਰਦਾ ਹੈ। ਤੁਹਾਨੂੰ ਔਜ਼ਾਰ ਕਾਲ ਦਰਸਾਉਂਦਾ ਨਤੀਜਾ ਦੇਖਾਈ ਦੇਣਾ ਚਾਹੀਦਾ ਹੈ (ਉਦਾਹਰਨ ਲਈ, "Calling tool: add") ਅਤੇ ਉਸ ਕਾਲ ਦਾ ਨਤੀਜਾ।
