## ਸ਼ੁਰੂਆਤ ਕਰਨਾ  

[![ਆਪਣਾ ਪਹਿਲਾ MCP ਸਰਵਰ ਬਣਾਓ](../../../translated_images/pa/04.0ea920069efd979a.webp)](https://youtu.be/sNDZO9N4m9Y)

_(ਇਸ ਪਾਠ ਦੀ ਵੀਡੀਓ ਦੇਖਣ ਲਈ ਉਪਰੀ ਚਿੱਤਰ ਤੇ ਕਲਿੱਕ ਕਰੋ)_

ਇਹ ਭਾਗ ਕਈ ਪਾਠਾਂ 'ਤੇ مشتمل ਹੈ:

- **1 ਤੁਹਾਡਾ ਪਹਿਲਾ ਸਰਵਰ**, ਇਸ ਪਹਿਲੇ ਪਾਠ ਵਿੱਚ, ਤੁਸੀਂ ਆਪਣੇ ਪਹਿਲੇ ਸਰਵਰ ਨੂੰ ਬਣਾਉਣ ਅਤੇ ਇੰਸਪੈਕਟਰ ਟੂਲ ਨਾਲ ਜਾਂਚ ਕਰਨ ਬਾਰੇ ਸਿੱਖੋਗੇ, ਜੋ ਤੁਹਾਡੇ ਸਰਵਰ ਦਾ ਟੈਸਟ ਅਤੇ ਡੀਬੱਗ ਕਰਨ ਲਈ ਇੱਕ ਮਹੱਤਵਪੂਰਨ ਵਿਧੀ ਹੈ, [ਪਾਠ ਲਈ](01-first-server/README.md)

- **2 ਕਲਾਇੰਟ**, ਇਸ ਪਾਠ ਵਿੱਚ, ਤੁਸੀਂ ਸਿੱਖੋਗੇ ਕਿ ਕਿਵੇਂ ਇੱਕ ਕਲਾਇੰਟ ਲਿਖਣਾ ਹੈ ਜੋ ਤੁਹਾਡੇ ਸਰਵਰ ਨਾਲ ਜੁੜ ਸਕਦਾ ਹੈ, [ਪਾਠ ਲਈ](02-client/README.md)

- **3 ਕਲਾਇੰਟ LLM ਨਾਲ**, ਇੱਕ ਬਿਹਤਰ ਕਲਾਇੰਟ ਲਿਖਣ ਦਾ ਤਰੀਕਾ ਹੈ ਕਿ ਉਸ ਵਿੱਚ LLM ਜੋੜਿਆ ਜਾਵੇ ਤਾਂ ਜੋ ਉਹ ਤੁਹਾਡੇ ਸਰਵਰ ਨਾਲ "ਮੁਲਾਕਾਤ" ਕਰ ਸਕੇ ਕਿ ਕੀ ਕਰਣਾ ਹੈ, [ਪਾਠ ਲਈ](03-llm-client/README.md)

- **4 ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ ਵਿੱਚ ਗਿਟਹੱਬ ਕੋਪਾਈਲਟ ਏਜੰਟ ਮੋਡ ਵਿੱਚ ਸਰਵਰ ਦੀ ਵਰਤੋਂ**। ਇਥੇ, ਅਸੀਂ ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ ਤੋਂ ਸਾਡਾ MCP ਸਰਵਰ ਚਲਾਉਣ ਦੀ ਗੱਲ ਕਰ ਰਹੇ ਹਾਂ, [ਪਾਠ ਲਈ](04-vscode/README.md)

- **5 stdio ਟਰਾਂਸਪੋਰਟ ਸਰਵਰ** stdio ਟਰਾਂਸਪੋਰਟ ਸਥਾਨਕ MCP ਸਰਵਰ-ਤੋਂ-ਕਲਾਇੰਟ ਸੰਚਾਰ ਲਈ ਸੁਪਾਰੀਟੀਦਾਰ ਮਿਆਰ ਹੈ, ਜੋ ਸੁਰੱਖਿਅਤ subprocess-ਆਧਾਰਿਤ ਸੰਚਾਰ ਅਤੇ ਲਾਗੂ ਪ੍ਰਕਿਰਿਆ ਵਿਭਾਜਨ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ [ਪਾਠ ਲਈ](05-stdio-server/README.md)

- **6 MCP ਨਾਲ HTTP ਸਟ੍ਰੀਮਿੰਗ (ਸਟ੍ਰੀਮਯੋਗ HTTP)**. ਆਧੁਨਿਕ HTTP ਸਟ੍ਰੀਮਿੰਗ ਟਰਾਂਸਪੋਰਟ ਬਾਰੇ ਸਿੱਖੋ (ਮਾਲੁਮ MCP ਸਰਵਰਾਂ ਲਈ ਸਿਫਾਰਸ਼ ਕੀਤੀ ਗਈ ਵਿਧੀ [MCP Specification 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/basic/transports/#streamable-http)), ਪ੍ਰਗਟਨ ਸੂਚਨਾਵਾਂ ਅਤੇ ਕਿਵੇਂ ਸਟ੍ਰੀਮਯੋਗ HTTP ਦੀ ਵਰਤੋਂ ਨਾਲ ਵਿਸਤਾਰਯੋਗ, ਤਤਕਾਲ MCP ਸਰਵਰ ਅਤੇ ਕਲਾਇੰਟ ਬਣਾਏ ਜਾਣ, [ਪਾਠ ਲਈ](06-http-streaming/README.md)

- **7 VSCode ਲਈ AI ਟੂਲਕਿਟ ਦੀ ਵਰਤੋਂ** MCP ਕਲਾਇੰਟ ਅਤੇ ਸਰਵਰ ਦੀ ਵਰਤੋਂ ਅਤੇ ਟੈਸਟ ਕਰਨ ਲਈ [ਪਾਠ ਲਈ](07-aitk/README.md)

- **8 ਟੈਸਟਿੰਗ**. ਇੱਥੇ ਅਸੀਂ ਮੁੱਖ ਤੌਰ 'ਤੇ ਕਿਵੇਂ ਵੱਖ-ਵੱਖ ਤਰੀਕਿਆਂ ਨਾਲ ਆਪਣੇ ਸਰਵਰ ਅਤੇ ਕਲਾਇੰਟ ਦੀ ਟੈਸਟਿੰਗ ਕਰ ਸਕਦੇ ਹਾਂ, 'ਤੇ ਧਿਆਨ ਦੇਵਾਂਗੇ, [ਪਾਠ ਲਈ](08-testing/README.md)

- **9 ਡਿਪਲੋਯਮੈਂਟ**. ਇਸ ਅਧਿਆਯ ਵਿੱਚ ਤੁਹਾਡੇ MCP ਹੱਲਾਂ ਨੂੰ ਵੱਖ-ਵੱਖ ਤਰੀਕਿਆਂ ਨਾਲ ਡਿਪਲੋਯ ਕਰਨ ਬਾਰੇ ਗੱਲ ਕਰਾਂਗੇ, [ਪਾਠ ਲਈ](09-deployment/README.md)

- **10 ਉੱਨਤ ਸਰਵਰ ਵਰਤੋਂ**. ਇਸ ਅਧਿਆਯ ਵਿੱਚ ਉੱਨਤ ਸਰਵਰ ਵਰਤੋਂ ਨੂੰ ਛੇੜਿਆ ਗਿਆ ਹੈ, [ਪਾਠ ਲਈ](./10-advanced/README.md)

- **11 ਪ੍ਰਮਾਣੀਕਰਨ**. ਇਹ ਅਧਿਆਯ ਸਰਲ ਪ੍ਰਮਾਣੀਕਰਨ ਨੂੰ ਕਵਰ ਕਰਦਾ ਹੈ, ਬੇਸਿਕ ਪ੍ਰਮਾਣੀਕਰਨ ਤੋਂ ਲੈ ਕੇ JWT ਅਤੇ RBAC ਦੀ ਵਰਤੋਂ ਤੱਕ। ਤੁਹਾਨੂੰ ਇੱਥੇ ਸ਼ੁਰੂ ਕਰਨ ਲਈ ਪ੍ਰੇਰਿਤ ਕੀਤਾ ਜਾਂਦਾ ਹੈ ਅਤੇ ਫਿਰ ਚੈਪਟਰ 5 ਵਿੱਚ ਉੱਨਤ ਵਿਸ਼ਿਆਂ ਨੂੰ ਦੇਖੋ ਅਤੇ ਚੈਪਟਰ 2 ਵਿੱਚ ਸਿਫ਼ਾਰਸ਼ਾਂ ਰਾਹੀਂ ਵਾਧੂ ਸੁਰੱਖਿਆ ਸਖਤੀ ਕਰਨ ਦੀ ਕਰਵਾਈ ਕਰੋ, [ਪਾਠ ਲਈ](./11-simple-auth/README.md)

- **12 MCP ਹੋਸਟਸ**. ਲੋਕ-ਪ੍ਰਸਿੱਧ MCP ਹੋਸਟ ਕਲਾਇੰਟਾਂ ਨੂੰ ਕਨਫਿਗਰ ਅਤੇ ਵਰਤੋਂ ਕਰੋ ਜਿਵੇਂ Claude ਡੈਸਕਟਾਪ, Cursor, Cline, ਅਤੇ Windsurf. ਟਰਾਂਸਪੋਰਟ ਕਿਸਮਾਂ ਅਤੇ ਸਮੱਸਿਆ-ਹੱਲ ਬਾਰੇ ਜਾਣੋ, [ਪਾਠ ਲਈ](./12-mcp-hosts/README.md)

- **13 MCP ਇੰਸਪੈਕਟਰ**. MCP ਇੰਸਪੈਕਟਰ ਟੂਲ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਆਪਣੇ MCP ਸਰਵਰਾਂ ਨੂੰ ਇੰਟਰੈਕਟਿਵ ਤੌਰ 'ਤੇ ਡੀਬੱਗ ਅਤੇ ਟੈਸਟ ਕਰੋ। ਸਮੱਸਿਆ-ਹੱਲ ਟੂਲ, ਸਰੋਤ ਅਤੇ ਪ੍ਰੋਟੋਕੋਲ ਸੁਨੇਹਿਆਂ ਨੂੰ ਸਿੱਖੋ, [ਪਾਠ ਲਈ](./13-mcp-inspector/README.md)

- **14 ਨਮੂਨਾਂ ਦੀ ਬਣਤਰ**. ਐਸੇ MCP ਸਰਵਰ ਬਣਾਓ ਜੋ MCP ਕਲਾਇੰਟਾਂ ਨਾਲ LLM ਸੰਬੰਧੀ ਕੰਮਾਂ ਤੇ ਸਹਿਯੋਗ ਕਰਦੇ ਹਨ (`2026-07-28` ਰਿਲੀਜ਼ ਉਮੀਦਵਾਰ ਵਿੱਚ ਮੁਆਵਜ਼ਾ ਹੈ; ਤਬ ਵੀ `2025-11-25` ਲਈ ਵੀਧ ਹੈ) [ਪਾਠ ਲਈ](./14-sampling/README.md)

- **15 MCP ਐਪਸ**. ਐਸੇ MCP ਸਰਵਰ ਬਣਾਓ ਜੋ UI ਹਦਾਇਤਾਂ ਨਾਲ ਵੀ ਜਵਾਬ ਦੇਂਦੇ ਹਨ, [ਪਾਠ ਲਈ](./15-mcp-apps/README.md)

ਮਾਡਲ ਕੰਟੈਕਸਟ ਪ੍ਰੋਟੋਕੋਲ (MCP) ਇੱਕ ਖੁੱਲਾ ਪ੍ਰੋਟੋਕੋਲ ਹੈ ਜੋ ਇਹ ਮਿਆਰੀਕਰਦਾ ਹੈ ਕਿ ਐਪਲੀਕੇਸ਼ਨਾਂ ਕਿਸ ਤਰ੍ਹਾਂ LLMs ਨੂੰ ਪ੍ਰਸੰਗ ਦਿੰਦੇ ਹਨ। MCP ਨੂੰ AI ਐਪਲੀਕੇਸ਼ਨਾਂ ਲਈ ਇੱਕ USB-C ਪੋਰਟ ਵਾਂਗ ਸੋਚੋ - ਇਹ ਵੱਖ-ਵੱਖ ਡਾਟਾ ਸਰੋਤਾਂ ਅਤੇ ਟੂਲਾਂ ਨਾਲ AI ਮਾਡਲਾਂ ਨੂੰ ਜੋੜਨ ਦਾ ਮਿਆਰੀ ਤਰੀਕਾ ਦਿੰਦਾ ਹੈ।

## ਸਿੱਖਣ ਦੇ ਮਕਸਦ

ਇਸ ਪਾਠ ਦੇ ਆਖිර ਵਿੱਚ, ਤੁਸੀਂ ਸਮਰੱਥ ਹੋਵੋਗੇ:

- MCP ਲਈ C#, ਜਾਵਾ, ਪਾਈਥਨ, ਟਾਈਪਸਕ੍ਰਿਪਟ ਅਤੇ ਜਾਵਾਸਕ੍ਰਿਪਟ ਵਿੱਚ ਵਿਕਾਸ ਵਾਤਾਵਰਨ ਸੈੱਟ ਅੱਪ ਕਰਨਾ
- ਮੂਲ MCP ਸਰਵਰ ਕਸਟਮ ਫੀਚਰਾਂ (ਸਰੋਤ, ਪ੍ਰੋਮਪਟ ਅਤੇ ਟੂਲ) ਨਾਲ ਬਣਾਉਣਾ ਅਤੇ ਡਿਪਲੋਯ ਕਰਨਾ
- MCP ਸਰਵਰਾਂ ਨਾਲ ਜੁੜਨ ਵਾਲੀਆਂ ਹੋਸਟ ਐਪਲੀਕੇਸ਼ਨਾਂ ਬਣਾਉਣਾ
- MCP ਇੰਪਲੀਮੈਂਟੇਸ਼ਨਾਂ ਦੀ ਟੈਸਟਿੰਗ ਅਤੇ ਡੀਬੱਗਿੰਗ ਕਰਨਾ
- ਆਮ ਸੈੱਟਅੱਪ ਚੁਣੌਤੀਆਂ ਅਤੇ ਉਨ੍ਹਾਂ ਦੇ ਸਮਾਧਾਨਾਂ ਨੂੰ ਸਮਝਣਾ
- ਆਪਣੀਆਂ MCP ਇੰਪਲੀਮੈਂਟੇਸ਼ਨਾਂ ਨੂੰ ਲੋਕਪ੍ਰਿਯ LLM ਸੇਵਾਵਾਂ ਨਾਲ ਜੋੜਨਾ

## ਆਪਣੇ MCP ਵਾਤਾਵਰਨ ਦੀ ਸੈਟਅੱਪ ਕਰਨਾ

MCP ਨਾਲ ਕੰਮ ਸ਼ੁਰੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ, ਆਪਣਾ ਵਿਕਾਸ ਵਾਤਾਵਰਨ ਤਿਆਰ ਕਰਨਾ ਅਤੇ ਮੁਢਲੀ ਕਾਰਜਪ੍ਰਧਾਤਮਿਕਤਾ ਨੂੰ ਸਮਝਣਾ ਜਰੂਰੀ ਹੈ। ਇਹ ਭਾਗ MCP ਨਾਲ ਸੁਚਾਰੂ ਸ਼ੁਰੂਆਤ ਲਈ ਮੁੱਢਲੇ ਸੈਟਅੱਪ ਕਦਮਾਂ ਦੀ ਮਦਦ ਕਰੇਗਾ।

### ਲੋੜੀਂਦੀਆਂ ਚੀਜ਼ਾਂ

MCP ਵਿਕਾਸ ਵਿੱਚ ਡੁਬਕੀ ਲਗਾਉਣ ਤੋਂ ਪਹਿਲਾਂ ਇਹ ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡੇ ਕੋਲ ਹਨ:

- **ਵਿਕਾਸ ਵਾਤਾਵਰਨ**: ਆਪਣੇ ਚੁਣੇ ਭਾਸ਼ਾ ਲਈ (C#, ਜਾਵਾ, ਪਾਈਥਨ, ਟਾਈਪਸਕ੍ਰਿਪਟ, ਜਾਂ ਜਾਵਾਸਕ੍ਰਿਪਟ)
- **IDE/ਸੰਪਾਦਕ**: ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ, ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ, ਇੰਟੈਲੀਜੇ ਆਈਡੀਈ, ਇਕਲਿਪਸ, ਪਾਈਚਾਰਮ ਜਾਂ ਕੋਈ ਵੀ ਆਧੁਨਿਕ ਕੋਡ ਸੰਪਾਦਕ
- **ਪੈਕੇਜ ਮੈਨੇਜਰ**: NuGet, Maven/Gradle, pip, ਜਾਂ npm/yarn
- **API ਕੁੰਜੀਆਂ**: ਜੇਕਰ ਤੁਸੀਂ ਮਜ਼ਬੂਤ ਹੋਸਟ ਐਪਲੀਕੇਸ਼ਨਾਂ ਵਿੱਚ ਕਿਸੇ AI ਸੇਵਾਵਾਂ ਦੀ ਯੋਜਨਾ ਕਰਦੇ ਹੋ


### ਅਧਿਕਾਰਿਤ SDKs

ਅਗਲੇ ਚੈਪਟਰਾਂ ਵਿੱਚ ਤੁਸੀਂ ਪਾਈਥਨ, ਟਾਈਪਸਕ੍ਰਿਪਟ, ਜਾਵਾ ਅਤੇ .NET ਦੀ ਵਰਤੋਂ ਨਾਲ ਬਣਾਏ ਹੱਲ ਵੇਖੋਗੇ। ਇੱਥੇ ਸਾਰੇ ਅਧਿਕਾਰਿਤ SDKs ਹਨ।

MCP ਵੱਖ-ਵੱਖ ਭਾਸ਼ਾਵਾਂ ਲਈ ਅਧਿਕਾਰਿਤ SDK ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ (ਮੈਚਡ MCP ਵਿਸ਼ੇਸ਼ਤਾ 2025-11-25 ਨਾਲ): 
- [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk) - ਮਾਇਕਰੋਸਾਫਟ ਨਾਲ ਸਹਿਯੋਗ ਵਿੱਚ ਸੰਭਾਲਿਆ ਗਿਆ
- [ਜਾਵਾ SDK](https://github.com/modelcontextprotocol/java-sdk) - ਸਪ੍ਰਿੰਗ AI ਨਾਲ ਸਹਿਯੋਗ ਵਿੱਚ ਸੰਭਾਲਿਆ ਗਿਆ
- [ਟਾਈਪਸਕ੍ਰਿਪਟ SDK](https://github.com/modelcontextprotocol/typescript-sdk) - ਅਧਿਕਾਰਿਤ ਟਾਈਪਸਕ੍ਰਿਪਟ ਅਮਲਦਾਰੀ
- [ਪਾਈਥਨ SDK](https://github.com/modelcontextprotocol/python-sdk) - ਅਧਿਕਾਰਿਤ ਪਾਈਥਨ ਅਮਲਦਾਰੀ (FastMCP)
- [ਕੋਟਲਿਨ SDK](https://github.com/modelcontextprotocol/kotlin-sdk) - ਅਧਿਕਾਰਿਤ ਕੋਟਲਿਨ ਅਮਲਦਾਰੀ
- [ਸਵਿਫਟ SDK](https://github.com/modelcontextprotocol/swift-sdk) - ਲੂਪਵਰਕ AI ਨਾਲ ਸਹਿਯੋਗ ਵਿੱਚ ਸੰਭਾਲਿਆ ਗਿਆ
- [ਰਸਟ SDK](https://github.com/modelcontextprotocol/rust-sdk) - ਅਧਿਕਾਰਿਤ ਰਸਟ ਅਮਲਦਾਰੀ
- [ਗੋ SDK](https://github.com/modelcontextprotocol/go-sdk) - ਅਧਿਕਾਰਿਤ ਗੋ ਅਮਲਦਾਰੀ

## ਮੁੱਖ ਗੱਲਾਂ

- MCP ਵਿਕਾਸ ਵਾਤਾਵਰਨ ਸੈੱਟਅੱਪ ਭਾਸ਼ਾ-ਵਿਸ਼ੇਸ਼ SDKs ਨਾਲ ਆਸਾਨ ਹੈ
- MCP ਸਰਵਰ ਬਣਾਉਣਾ ਚੀਜ਼ਾਂ ਨੂੰ ਬਣਾਉਣਾ ਅਤੇ ਉਹਨਾਂ ਨੂੰ ਸਪੱਸ਼ਟ ਸਕੀਮਾਂ ਸਹਿਤ ਰਜਿਸਟਰ ਕਰਨਾ ਸ਼ਾਮਲ ਹੈ
- MCP ਕਲਾਇੰਟ ਸਰਵਰਾਂ ਅਤੇ ਮਾਡਲਾਂ ਨਾਲ ਜੁੜਦੇ ਹਨ ਤਾਂ ਕਿ ਵਾਧੂ ਸਮਰੱਥਾਵਾਂ ਦਾ ਲਾਭ ਲੈ ਸਕਣ
- ਟੈਸਟਿੰਗ ਅਤੇ ਡੀਬੱਗਿੰਗ MCP ਦੀ ਭਰੋਸੇਮੰਦ ਲਾਗੂਆਤ ਲਈ ਜ਼ਰੂਰੀ ਹਨ
