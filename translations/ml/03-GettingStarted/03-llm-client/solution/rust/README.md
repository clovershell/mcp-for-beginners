# ഈ സാമ്പിൾ റൺ ചെയ്യുന്നു

ഇത് LLM ക്ലയന്റ് സാമ്പിളിനുള്ള റസ്റ്റ് പരിഹാരമാണ്. നിങ്ങൾക്ക് റസ്റ്റ് ടൂൾചെയിൻ ഇൻസ്റ്റാൾ ചെയ്തിരിക്കണം; [അധികാരিক ഇൻസ്റ്റാൾ ഗൈഡ്](https://www.rust-lang.org/tools/install) കാണുക.

ക്ലയന്റ് GitHub മോഡൽസ് ഇൻഫറൻസ് എൻഡ്പോയിന്റായ (`https://models.github.ai/inference/chat`) വഴിയാണ് മോഡലിനെ വിളിക്കുന്നത്, കൂടാതെ `OPENAI_API_KEY` എൻവയ്റൺമെന്റ് വേരിയബിളിൽ നിന്നു നിങ്ങളുടെ GitHub വ്യക്തിഗത ആക്സസ് ടോക്കൺ (PAT) വായിക്കുന്നു.

> [!NOTE]
> ഈ റിപോസിറ്ററിയിലെ മറ്റ് പരിഹാരങ്ങൾ `GITHUB_TOKEN` ഉപയോഗിക്കുന്നു. Rust-ിനായി, OpenAI ക്ലയന്റ് കോൺഫിഗറേഷനുമായി പൊരുത്തപ്പെടാൻ `OPENAI_API_KEY` യേ സംവരണ മൂല്യമാക്കുക.

## -0- നിങ്ങളുടെ GitHub ടോക്കൺ സെറ്റ് ചെയ്യുക

```bash
# zsh/bash
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# പവർഷെൽ
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- സാമ്പിൾ ടെസ്റ്റ് നിർമാണം

```bash
cargo build
```

## -2- സാമ്പിൾ റൺ ചെയ്യുക

```bash
cargo run
```

ക്ലയന്റ് കാൽക്കുലേറ്റർ MCP സർവർ ആരംഭിക്കുന്നു, ടൂൾ ലിസ്റ്റ് കൊണ്ടുവരുന്നു, പിന്നീട് മോഡൽ (`openai/gpt-5-mini`) ഉപയോഗിച്ച് `add` ടൂൾ വിളിക്കുന്നു. ഇതിന്റെ ഫലം കാണുന്നുണ്ടാകും (ഉദാഹരണത്തിന്, "Calling tool: add" പോലുള്ള മെസ്സേജ്).
