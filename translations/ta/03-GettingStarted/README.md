## துவக்கம்  

[![Build Your First MCP Server](../../../translated_images/ta/04.0ea920069efd979a.webp)](https://youtu.be/sNDZO9N4m9Y)

_(இந்த பாடத்தின் வீடியோவை பார்க்க மேலே உள்ள படத்தை கிளிக் செய்யவும்)_

இந்த பகுதி பல பாடங்களைக் கொண்டுள்ளது:

- **1 உங்கள் முதல் சர்வர்**, இந்த முதல் பாடத்தில், நீங்கள் உங்கள் முதல் சர்வரை உருவாக்கி, அதை இன்ஸ்பெக்டர் கருவியின் மூலம் பரீட்சிக்க மற்றும் பிழைத்திருத்த கற்றுக்கொள்பீர்கள், [பாடத்திற்கு](01-first-server/README.md)

- **2 கிளையண்ட்**, இந்த பாடத்தில், நீங்கள் உங்கள் சர்வருடன் இணைக்கக்கூடிய கிளையண்டை எழுதுவது எப்படி என்பதை கற்றுக்கொள்பீர்கள், [பாடத்திற்கு](02-client/README.md)

- **3 LLM உடன் கிளையண்ட்**, சரியான முறையில் ஒரு கிளையண்டை எழுதுவதற்கான ஒரு சிறந்த வழி LLM ஐ அதில் சேர்ப்பது ஆகும், அதனால் அது உங்கள் சர்வருடன் "சம்பந்தப்படுத்த" முடியும், [பாடத்திற்கு](03-llm-client/README.md)

- **4 Visual Studio Code இல் GitHub Copilot ஏஜென்ட் முறையில் ஒரு சர்வரை பயன்படுத்துதல்**. இங்கு, நாம் Visual Studio Code உள்ளே இருந்து எமது MCP சர்வரை இயக்குவது பற்றி பார்க்கிறோம், [பாடத்திற்கு](04-vscode/README.md)

- **5 stdio டிரான்ஸ்போர்ட் சர்வர்** stdio டிரான்ஸ்போர்ட் என்பது உள்ளூர் MCP சர்வர்-கிளையண்ட் தொடர்புக்கு பரிந்துரைக்கபட்ட நிலையான முறையாகும், இது பாதுகாப்பான subprocess அடிப்படையிலான தொடர்பை வழங்குகிறது மற்றும் உட்பிரதான செயல்முறை தனிப்பட்ட தன்மையை கொண்டுள்ளது [பாடத்திற்கு](05-stdio-server/README.md)

- **6 MCP உடன் HTTP ஸ்டிரீமிங் (Streamable HTTP)**. நவீன HTTP ஸ்டிரீமிங் டிரான்ஸ்போர்ட் பற்றி (தொலை MCP சர்வர்களுக்கு பரிந்துரைக்கப்பட்டது [MCP Specification 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/basic/transports/#streamable-http) படி), முன்னேற்ற அறிவிப்புகள் மற்றும் எவ்வாறு ஸ்கேலபிள், நேரடி MCP சர்வர்களையும் கிளையண்ட்களையும் Streamable HTTP மூலம் செயல்படுத்துவது என்பதை கற்றுக்கொள்ளுங்கள். [பாடத்திற்கு](06-http-streaming/README.md)

- **7 VSCode க்கான AI கருவிகள் தொகுப்பைப் பயன்படுத்துதல்** MCP கிளையண்ட்கள் மற்றும் சர்வர்களை சாப்பிடவும் மற்றும் பரிசோதிக்கவும் [பாடத்திற்கு](07-aitk/README.md)

- **8 பரிசோதனை**. இங்கு நாங்கள் எங்கள் சர்வர் மற்றும் கிளையண்டை வெவ்வேறு முறைகளில் எப்படி பரிசோதிக்க முடியும் என்பதில் கவனம் செலுத்துகிறோம், [பாடத்திற்கு](08-testing/README.md)

- **9 பகிர்வு**. இங்கு உங்கள் MCP தீர்வுகளை பகிர்வதற்கான வெவ்வேறு வழிகளைப் பார்க்கிறோம், [பாடத்திற்கு](09-deployment/README.md)

- **10 மேம்பட்ட சர்வர் பயன்பாடு**. இந்த அத்தியாயம் மேம்பட்ட சர்வர் பயன்பாட்டை பற்றியது, [பாடத்திற்கு](./10-advanced/README.md)

- **11 அத்தரவு**. இந்த அத்தியாயம் எளிய அத்தரவைச் சேர்ப்பது பற்றி, மைய அர்த்த அத்தரவு முதல் JWT மற்றும் RBAC பயன்படுத்துதல் வரை கவர்ந்துள்ளது. நீங்கள் இங்கு தொடங்கி பின்னர் அத்தியாயம் 5 இல் மேம்பட்ட தலைப்புகளை பார்த்து, அத்தியாயம் 2 இல் பரிந்துரைக்கபட்ட கூடுதல் பாதுகாப்பு கட்டமைப்புகளை செய்ய வேண்டும், [பாடத்திற்கு](./11-simple-auth/README.md)

- **12 MCP ஹோஸ்ட்கள்**. Claude Desktop, Cursor, Cline மற்றும் Windsurf உட்பட பிரபலமான MCP ஹோஸ்ட் கிளையண்ட்களைக் கட்டமைத்து பயன்படுத்துங்கள். டிரான்ஸ்போர்ட் வகைகள் மற்றும் பிரச்சினைச் சரிசெய்தல்கள் பற்றி கற்றுக்கொள்ளுங்கள், [பாடத்திற்கு](./12-mcp-hosts/README.md)

- **13 MCP இன்ஸ்பெக்டர்**. MCP இன்ஸ்பெக்டர் கருவியைப் பயன்படுத்தி உங்கள் MCP சர்வர்களை இடையே இடைமுகமாய் பிழைத்திருத்தி பரிசோதியுங்கள். கருவிகள், வளங்கள் மற்றும் புரோட்டோகால் செய்திகளை சரிசெய்வது எப்படி என்பதைக் கற்றுக்கொள்ளுங்கள், [பாடத்திற்கு](./13-mcp-inspector/README.md)

- **14 சாம்பிளிங்**. LLM சம்பந்தப்பட்ட பணிகளில் MCP கிளையண்ட்களுடன் ஒத்துழைக்கும் MCP சர்வர்களை உருவாக்குங்கள் ( `2026-07-28` வெளியீட்டில் பழுதுபார்க்கப்பட்டது; இன்னும் `2025-11-25` இல் செல்லுபடியாகும்), [பாடத்திற்கு](./14-sampling/README.md)

- **15 MCP அப்ளிகேஷன்கள்**. UI வழிமுறைகளையும் பதிலளிக்கும் MCP சர்வர்களை உருவாக்குங்கள், [பாடத்திற்கு](./15-mcp-apps/README.md)

மாடல் கான்டெக்ஸ்ட் புரோட்டோக்கோல் (MCP) என்பது பயன்பாடுகள் எப்படி LLM களுக்கு முன்னொட்டு வழங்குகின்றன என்பது நிறுவனமடைய சீர்மையான ஓர் திறந்த புரோட்டோக்கோல் ஆகும். MCP ஐ AI பயன்பாடுகளுக்கான USB-C போர்ட்டாக நினைத்துக்கொள்ளுங்கள் - இது AI மாதிரிகளை பல்வேறு தரவுதளங்கள் மற்றும் கருவிகளுடன் இணைக்கும் நிலையான முறையை வழங்குகிறது.

## கற்றல் நோக்கங்கள்

இந்த பாடம் முடிந்தவுடன், நீங்கள் செய்யக்கூடியவை:

- C#, Java, Python, TypeScript மற்றும் JavaScript மொழிகளில் MCP க்கான மேம்பாட்டு சூழலை அமைக்க
- தனிப்பயன் அம்சங்களுடன் சாதாரண MCP சர்வர்களை உருவாக்கி பரவி செயல்படுத்த
- MCP சர்வர்களுடன் இணைகக்கூடிய ஹோஸ்ட் அப்ளிகேஷன்களை உருவாக்க
- MCP அமலாக்கங்களின் பரிசோதனை மற்றும் பிழைத்திருத்தலை செய்ய
- பொதுவான கட்டமைப்பு சவால்கள் மற்றும் அவற்றின் தீர்வுகளை புரிந்து கொள்
- MCP அமலாக்கங்களை பிரபலமான LLM சேவைகளுடன் இணைக்க

## உங்கள் MCP சூழலை அமைத்தல்

MCP உடன் பணியாற்றத் துவங்கும் முன்னர், உங்கள் மேம்பாட்டு சூழலை தயார் செய்வதும் அடிப்படை செயல் படிவத்தைக் கற்றுக்கொள்ளும் முக்கியத்துவம் உள்ளது. இந்த பகுதி MCP உடன் சீரான தொடக்கத்துக்கு தேவையான ஆரம்ப அமைப்பை வழிகாட்டும்.

### தேவையானவை

MCP மேம்பாட்டிற்க்குள் நிற்கும் முன்னர், கீழ்கண்டவற்றை உறுதி செய்யுங்கள்:

- **மேம்பாட்டு சூழல்**: நீங்கள் தேர்ந்தெடுத்த மொழிக்கான (C#, Java, Python, TypeScript அல்லது JavaScript)
- **IDE/எடிட்டர்**: Visual Studio, Visual Studio Code, IntelliJ, Eclipse, PyCharm அல்லது எதையும் சமகால குறியிடல் கருவிகள்
- **பேக்கேஜ் மேலாளர்கள்**: NuGet, Maven/Gradle, pip அல்லது npm/yarn
- **API விசைகள்**: நீங்கள் உங்கள் ஹோஸ்ட் அப்ளிகேஷன்களில் பயன்படுத்தப் போகும் எந்த AI சேவைகளுக்கும்


### அதிகாரப்பூர்வ SDK கள்

வரும் அத்தியாயங்களில் Python, TypeScript, Java மற்றும் .NET பயன்படுத்தி கட்டப்பட்ட தீர்வுகளை நீங்கள் காண்பீர்கள். இங்கே அதிகாரப்பூர்வமாக ஆதரிக்கப்படும் அனைத்து SDK களும் உள்ளன.

MCP பல மொழிகளுக்கான அதிகாரப்பூர்வ SDK களை வழங்குகிறது ([MCP Specification 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/) படி):
- [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk) - Microsoft உடன் ஒத்துழைப்பில் பராமரிக்கப்படுகிறது
- [Java SDK](https://github.com/modelcontextprotocol/java-sdk) - Spring AI உடன் ஒத்துழைப்பில் பராமரிக்கப்படுகிறது
- [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) - அதிகாரப்பூர்வ TypeScript செயலாக்கம்
- [Python SDK](https://github.com/modelcontextprotocol/python-sdk) - அதிகாரப்பூர்வ Python செயலாக்கம் (FastMCP)
- [Kotlin SDK](https://github.com/modelcontextprotocol/kotlin-sdk) - அதிகாரப்பூர்வ Kotlin செயலாக்கம்
- [Swift SDK](https://github.com/modelcontextprotocol/swift-sdk) - Loopwork AI உடன் ஒத்துழைப்பில் பராமரிக்கப்படுகிறது
- [Rust SDK](https://github.com/modelcontextprotocol/rust-sdk) - அதிகாரப்பூர்வ Rust செயலாக்கம்
- [Go SDK](https://github.com/modelcontextprotocol/go-sdk) - அதிகாரப்பூர்வ Go செயலாக்கம்

## முக்கிய கற்கை அம்சங்கள்

- மொழி-நிர్దಿಷ್ಟ SDK களுடன் MCP மேம்பாட்டு சூழலை அமைத்தல் எளிது
- சுத்தமான திட்டங்களுடன் கருவிகளை உருவாக்கி பதிவு செய்வதும் MCP சர்வர்களை கட்டுவது முக்கியம்
- MCP கிளையண்ட்கள் சர்வர் மற்றும் மாதிரிகளுடன் இணைத்து விரிவாக்கப்பட்ட திறன்களைப் பயன்படுத்துகின்றன
- பரிசோதனை மற்றும் பிழைத்திருத்தல் நம்பகமான MCP செயலாக்கங்களுக்கு அவசியம்
- பகிர்வு விருப்பங்கள் உள்ளூர் மேம்பாட்டில் இருந்து மேக அடிப்படையில் தீர்வுகள் வரை வகைப்படுத்தப்படுகின்றன
