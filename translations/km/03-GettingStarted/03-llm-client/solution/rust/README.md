# កំពុងរត់ឧទាហរណ៍នេះ

នេះគឺជាជម្រើស Rust សម្រាប់ឧទាហរណ៍អ្នកប្រើ LLM។ អ្នកត្រូវការតម្លើង toolchain Rust មួយ; មើល [មាគ៌ាបង្ហាញដំឡើងផ្លូវការជា](https://www.rust-lang.org/tools/install)។

អ្នកប្រើហៅម៉ូដែលតាមរយៈចំណុចផុសប្រាប់ GitHub Models inference (`https://models.github.ai/inference/chat`) ហើយអានសម្បក្ដិផ្ទាល់ខ្លួន GitHub របស់អ្នក (PAT) ពីអថេរបរិស្ថាន `OPENAI_API_KEY`។

> [!NOTE]
> ជម្រើសផ្សេងទៀតនៅក្នុងហាងនេះប្រើ `GITHUB_TOKEN`។ សម្រាប់ Rust សូមកំណត់ `OPENAI_API_KEY` ទៅតម្លៃដូចគ្នាដើម្បីផ្គូផ្គងការកំណត់រចនាសម្ព័ន្ធអ្នកប្រើ OpenAI។

## -0- កំណត់សម្បក្ស GitHub របស់អ្នក

```bash
# zsh/bash
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# បាវើសែល
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- សាងសង់ឧទាហរណ៍

```bash
cargo build
```

## -2- ប្រតិបត្តិឧទាហរណ៍

```bash
cargo run
```

អ្នកប្រើចាប់ផ្តើមម៉ាស៊ីនបម្រើ calculator MCP, ទាញយកបញ្ជីឧបករណ៍របស់វា ហើយប្រើម៉ូដែល (`openai/gpt-5-mini`) ដើម្បីហៅឧបករណ៍ `add`។ អ្នកគួរតែឃើញលទ្ធផលសម្គាល់ការហៅឧបករណ៍ (ឧទាហរណ៍ "Calling tool: add") និងលទ្ធផលនៃការហៅនោះ។
