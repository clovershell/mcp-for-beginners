# Запуск цього прикладу

Це рішення на Rust для клієнта LLM. Вам потрібно встановити тулчейн Rust; див. [офіційний посібник з установки](https://www.rust-lang.org/tools/install).

Клієнт виконує виклик моделі через кінцеву точку GitHub Models inference (`https://models.github.ai/inference/chat`) і зчитує ваш особистий токен доступу GitHub (PAT) з змінної середовища `OPENAI_API_KEY`.

> [!NOTE]
> Інші рішення у цьому репозиторії використовують `GITHUB_TOKEN`. Для Rust встановіть `OPENAI_API_KEY` у те саме значення, щоб відповідати конфігурації клієнта OpenAI.

## -0- Встановіть ваш токен GitHub

```bash
# zsh/bash
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# PowerShell
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- Зберіть приклад

```bash
cargo build
```

## -2- Запустіть приклад

```bash
cargo run
```

Клієнт запускає сервер калькулятора MCP, отримує список його інструментів і використовує модель (`openai/gpt-5-mini`) для виклику інструменту `add`. Ви маєте побачити вивід, що вказує на виклик інструменту (наприклад, "Calling tool: add") та результат цього виклику.
