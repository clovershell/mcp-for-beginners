# এই উদাহরণ চালানো

এটি LLM ক্লায়েন্ট নমুনার জন্য রাষ্ট সমাধান। আপনার কাছে একটি রাষ্ট টুলচেইন ইনস্টল থাকতে হবে; দেখুন [আধিকারিক ইনস্টল গাইড](https://www.rust-lang.org/tools/install)।

ক্লায়েন্টটি GitHub Models inference endpoint (`https://models.github.ai/inference/chat`) এর মাধ্যমে একটি মডেল কল করে এবং আপনার GitHub পার্সোনাল অ্যাক্সেস টোকেন (PAT) পড়ে `OPENAI_API_KEY` পরিবেশ চলক থেকে।

> [!NOTE]
> এই রেপোতে অন্য সমাধানগুলি `GITHUB_TOKEN` ব্যবহার করে। রাষ্ট এর জন্য, OpenAI ক্লায়েন্ট কনফিগারেশনের সাথে মেলাতে `OPENAI_API_KEY` একই মানে সেট করুন।

## -0- আপনার GitHub টোকেন সেট করুন

```bash
# জেডএসএইচ/বাশ
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# পাওয়ারশেল
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- নমুনাটি তৈরি করুন

```bash
cargo build
```

## -2- নমুনাটি চালান

```bash
cargo run
```

ক্লায়েন্টটি ক্যালকুলেটর MCP সার্ভার শুরু করে, এর টুল তালিকা নিয়ে আসে, এবং মডেল (`openai/gpt-5-mini`) ব্যবহার করে `add` টুলটি কল করে। আপনি দেখতে পাবেন আউটপুট যা টুল কল নির্দেশ করে (উদাহরণস্বরূপ, "Calling tool: add") এবং ঐ কলের ফলাফল।
