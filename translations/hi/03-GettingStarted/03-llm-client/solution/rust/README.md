# इस सैंपल को चलाना

यह LLM क्लाइंट सैंपल के लिए Rust समाधान है। आपको Rust टूलचेन स्थापित करना होगा; देखें [अधिकारिक इंस्टॉल गाइड](https://www.rust-lang.org/tools/install)।

क्लाइंट GitHub Models इनफेरेंस एंडपॉइंट (`https://models.github.ai/inference/chat`) के माध्यम से मॉडल को कॉल करता है और आपके GitHub व्यक्तिगत पहुंच टोकन (PAT) को `OPENAI_API_KEY` वातावरण चर से पढ़ता है।

> [!NOTE]
> इस रिपो में अन्य समाधान `GITHUB_TOKEN` का उपयोग करते हैं। Rust के लिए, OpenAI क्लाइंट कॉन्फ़िगरेशन से मेल खाने के लिए `OPENAI_API_KEY` को उसी मान पर सेट करें।

## -0- अपना GitHub टोकन सेट करें

```bash
# zsh/bash
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# पावरशेल
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- सैंपल बनाएं

```bash
cargo build
```

## -2- सैंपल चलाएं

```bash
cargo run
```

क्लाइंट कैलकुलेटर MCP सर्वर शुरू करता है, उसके टूल की सूची प्राप्त करता है, और मॉडल (`openai/gpt-5-mini`) का उपयोग करके `add` टूल को कॉल करता है। आपको टूल कॉल का आउटपुट देखना चाहिए (उदाहरण के लिए, "Calling tool: add") और उस कॉल का परिणाम।
