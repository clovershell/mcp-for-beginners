# Запуск этого примера

Это решение на Rust для примера клиента LLM. Вам нужна установленная среда Rust; смотрите [официальное руководство по установке](https://www.rust-lang.org/tools/install).

Клиент вызывает модель через конечную точку вывода GitHub Models (`https://models.github.ai/inference/chat`) и читает ваш персональный токен доступа GitHub (PAT) из переменной окружения `OPENAI_API_KEY`.

> [!NOTE]
> Другие решения в этом репозитории используют `GITHUB_TOKEN`. Для Rust установите `OPENAI_API_KEY` в то же значение, чтобы соответствовать конфигурации клиента OpenAI.

## -0- Установите ваш токен GitHub

```bash
# zsh/bash
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# PowerShell
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- Соберите пример

```bash
cargo build
```

## -2- Запустите пример

```bash
cargo run
```

Клиент запускает сервер калькулятора MCP, получает список его инструментов и использует модель (`openai/gpt-5-mini`) для вызова инструмента `add`. Вы должны увидеть вывод, указывающий вызов инструмента (например, "Calling tool: add") и результат этого вызова.
