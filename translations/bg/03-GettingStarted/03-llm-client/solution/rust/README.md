# Стартиране на този пример

Това е Rust решение за примерния клиент LLM. Трябва да имате инсталиран Rust toolchain; вижте [официалното ръководство за инсталиране](https://www.rust-lang.org/tools/install).

Клиентът извиква модел чрез GitHub Models inference endpoint (`https://models.github.ai/inference/chat`) и чете вашия личен достъп токен (PAT) от променливата на средата `OPENAI_API_KEY`.

> [!NOTE]
> Други решения в това репо използват `GITHUB_TOKEN`. За Rust, задайте `OPENAI_API_KEY` със същата стойност, за да съвпадне с конфигурацията на OpenAI клиента.

## -0- Задайте вашия GitHub токен

```bash
# zsh/bash
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# PowerShell
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- Сглобете примера

```bash
cargo build
```

## -2- Стартирайте примера

```bash
cargo run
```

Клиентът стартира MCP сървъра за калкулатор, зарежда списъка с инструменти и използва модела (`openai/gpt-5-mini`) за извикване на инструмента `add`. Трябва да видите изход, показващ извикването на инструмента (например, "Calling tool: add") и резултата от това извикване.
