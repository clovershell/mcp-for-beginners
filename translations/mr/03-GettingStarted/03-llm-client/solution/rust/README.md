# हा नमुना चालवत आहे

हा LLM क्लायंट नमुन्यासाठी Rust सोल्यूशन आहे. तुम्हाला Rust टूलचेन स्थापित करावी लागेल; पाहा [अधिकृत इंस्टॉल मार्गदर्शिका](https://www.rust-lang.org/tools/install).

क्लायंट GitHub Models इन्फरेन्स एंडपॉइंट (`https://models.github.ai/inference/chat`) द्वारे मॉडेल कॉल करतो आणि तुमचा GitHub वैयक्तिक प्रवेश टोकन (PAT) `OPENAI_API_KEY` पर्यावरण चलातून वाचतो.

> [!NOTE]
> या रिपॉजिटरीमधील इतर सोल्यूशन्स `GITHUB_TOKEN` वापरतात. Rust साठी, OpenAI क्लायंट कॉन्फिगरेशनशी जुळण्यासाठी `OPENAI_API_KEY` ला त्याच मूल्यावर सेट करा.

## -0- आपला GitHub टोकन सेट करा

```bash
# झेडश/बॅश
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# पॉवरशेल
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- नमुना तयार करा

```bash
cargo build
```

## -2- नमुना चालवा

```bash
cargo run
```

क्लायंट कॅल्क्युलेटर MCP सर्व्हर सुरु करतो, त्याची टूल यादी आणतो, आणि `openai/gpt-5-mini` मॉडेल वापरुन `add` टूल कॉल करतो. तुम्हाला टूल कॉल दर्शवणारा आउटपुट (उदा. "Calling tool: add") आणि त्या कॉलचा निकाल दिसेल.
